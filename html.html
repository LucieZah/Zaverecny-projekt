HTML

HTML (HyperText Markup Language) je základní jazyk pro tvorbu webových stránek, který určuje strukturu a obsah stránky pomocí značek (tagů). Umožňuje přidávat text, obrázky, odkazy a další prvky které prohlížeče interpretují a zobrazují uživatelům.

Základní struktura stránky

Do prázdného souboru index.html zadej ! a stiskni Tab nebo Enter a editor kódu (např. Visual Studio Code) automaticky doplní základní HTML strukturu.

Tabulka

  <table>
      <tr> 
        <th>Text</th> 
        <th>Text</th>
      </tr> 
      <tr> 
        <td>Text</td> 
        <td>Text</td>
      </tr>
  </table>
  • tr = table row
  • th = table header
  • td = table data
