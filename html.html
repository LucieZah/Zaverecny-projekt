HTML

HTML (HyperText Markup Language) je základní jazyk pro tvorbu webových stránek, který určuje strukturu a obsah stránky pomocí značek (tagů). Umožňuje přidávat text, obrázky, odkazy a další prvky které prohlížeče interpretují a zobrazují uživatelům.

Základní struktura stránky

Do prázdného souboru index.html zadej ! a stiskni Tab nebo Enter a editor kódu (např. Visual Studio Code) automaticky doplní základní HTML strukturu.

Členění obsahu pomocí sémantických značek

Hlavička stránky <header> ..... </header>
Sekce <section> ..... </section>
Patička <footer> ..... </footer>

Nadpisy

<h1>Text nadpisu</h1>

Další úrovně nadpisů: <h2>, <h3>, <h4>, <h5>, <h6>

Odstavec

<p>Text odstavce</p>

Odkaz

<a href=”www.webovaadresa.cz”>Text odkazu</a>

Odkaz na mail

<a href=”mailto:mujmail@mail.com”>Text</a>

Zvýraznění

<strong>Text....</strong> v textu vypadá tučně

<em>Text....</em> v textu je kurzívou

Obrázky

<img src=“foto.jpg”> Zadej název obrázku včetně cesty (např. obrazky/foto.jpg).

<img src=“foto.jpg” alt="Podzimní les">

Tabulka

  <table>
    <tr> 
      <th>Text</th> 
      <th>Text</th>
    </tr> 
    <tr> 
      <td>Text</td> 
      <td>Text</td>
    </tr>
  </table>
  • tr = table row
  • th = table header
  • td = table data
