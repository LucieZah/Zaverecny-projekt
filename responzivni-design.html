Responzivní webdesign

Responzivní webdesign je přístup k tvorbě webových stránek, který zajišťuje, že obsah a rozložení stránky se automaticky přizpůsobí různým velikostem a typům zařízení (například mobilům, tabletům, notebookům a desktopům). Využívá technologie jako flexbox, grid a @media queries v CSS, aby zajistil správné zobrazení bez ohledu na rozlišení obrazovky. Cílem je, aby uživatelé měli pohodlný a funkční zážitek na všech zařízeních.

Media Query

Media Query je způsob, jak v CSS zařídit, abychom pro různé prostředí mohli prvku nastavit jiné CSS. Nejčastěji reagujeme na šířku okna prohlížeče/zařízení.

@media (min-width: 800px) { 
        css styly;
    }

Měli bychom také uvádět, pro jaký typ "média" naše query platí. Možné hodnoty: screen, print, speech, all (all je výchozí hodnota).

@media screen and (min-width: 800px){ 
        css styly;
    }

Lze používat kombinace:

@media screen and (min-width: 800px) and (max-width:1200px) and (orientation: landscape){ 
        css styly;
    }

Breakpoint

Breakpoint je bod, ve kterém dochází ke změně layoutu (rozložení) stránky. Breakpointy by se měly stanovovat podle obsahu, tedy změnit ve chvíli, kdy to pro daný obsah dává smysl.

Příklad využití breakpoitu ke změně vzhledu i umístění obsahu:

Zápis v HTML: 
    <div class="responsive__box">
    <div class="responsive__one">1</div>
    <div class="responsive__two">2</div>
    </div>
  
CSS:
    .responsive__one,
    .responsive__two {
        border: solid 2px var(--primary-color);
        padding: 50px;
        margin: 10px 0;
        font-size: 26px;
        background-color: #bad4f0;
    }

Pomocí Media Query můžeme rámečky na různých velikostech obrazovky umístit vedle sebe, změnit jejich velikost nebo třeba i barvu: 

    @media screen and (min-width:1000px){
        .responsive__box {
            display: flex;
            justify-content: space-evenly;
        }
        .responsive__one,
        .responsive__two {
            padding: 120px 150px;
            font-size: 40px;
            background-color: rgb(245, 149, 245);
            border: solid 50px violet;
            margin-right: 10px;
        }
    }
