CSS

CSS (Cascading Style Sheets) je jazyk pro stylování webových stránek, který umožňuje upravovat vzhled HTML elementů - například barvy, písma nebo rozložení.

Připojení CSS

CSS s HTML propoj tak, že na konci <head> v index.html zapiš:

<link rel=”stylesheet” href=”style.css”>

CSS

Syntaxe CSS je způsob, jakým se v CSS stylují HTML prvky. Skládá se ze tří částí:

Selektor - určuje, na které HTML elementy se daný styl aplikuje (např. h1, .class...)

Vlastnost - definuje, co chceme na elementu změnit (např. color, font-family...)

Hodnota - nastavuje konkrétní styl pro vlastnost (např. color: purple; font-family: roboto;...)

příklad zápisu:  
    h1 {
        color: blue;
        background-color: grey;
    }



Vícenásobný selektor:
    h1,
    a,
    .text {
        font-family: roboto;
     }

Selektor * vybírá všechny prvky na stránce. (Má nejnižší specificitu.)

Vlastnosti

Velikost písma font-size: 24px;
Výška řádku line-height: 30px; line-height: 2; (Doporučená hodnota pro běžný text je 1.5 - 1.8.)
Tloušťka písma font-weight: normal; (bold)
Sklon písma font-style: normal; (italic - kurzíva, oblique - skloněné písmo)
Písmo font-family: Roboto, Sans-Serif;
Zarovnání textu text-align: center; (left, right, justify)
Barva textu color: blueviolet; (#8826e3)
Barva pozadí background-color: aliceblue; (rgb(148, 164, 177))
Podtržení textu text-decoration: underline; (none, dotted...)
Margin margin: 20px;
Padding padding: 15px;
